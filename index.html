Front End Cheatsheet

Table Element

Tag Name Description
<table> Table Contains all tabular data. Wrapper element for HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data data container for each cell in the table.
<tfoot> Table Foot set of rows defining the footer in the table.
